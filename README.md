# 📋 Бот для сбора отчётов сотрудников

## Что умеет бот
- Собирает отчёты от сотрудников с выбором должности
- Спрашивает имя — анонимности нет
- Задаёт 4 вопроса по очереди
- Отправляет напоминание в 21:10
- Благодарит случайной фразой после отчёта
- Присылает тебе сводку в 09:00 следующего дня

---

## Шаг 1 — Создай бота в Telegram

1. Напиши @BotFather в Telegram
2. Отправь `/newbot`
3. Придумай имя и username (например `ShiftReportBot`)
4. Скопируй **токен** — он выглядит так: `7412345678:AAFxxxxxx`

---

## Шаг 2 — Узнай свой chat_id

1. Запусти бота (любое сообщение)
2. Напиши боту `/myid`
3. Скопируй число — это твой `MANAGER_CHAT_ID`

---

## Шаг 3 — Вставь данные в bot.py

Открой `bot.py` и замени:
```python
BOT_TOKEN = "ВАШ_ТОКЕН_ЗДЕСЬ"      # токен от BotFather
MANAGER_CHAT_ID = 123456789         # твой chat_id
TIMEZONE = "Europe/Moscow"          # или "Asia/Almaty", "Asia/Tashkent" и т.д.
```

---

## Шаг 4 — Запуск (выбери вариант)

### Вариант A: Бесплатно на Railway.app

1. Зарегистрируйся на https://railway.app (GitHub аккаунт)
2. New Project → Deploy from GitHub → загрузи папку с ботом
3. В переменных окружения добавь `BOT_TOKEN` и `MANAGER_CHAT_ID`
4. Бот работает 24/7 бесплатно (500 часов/месяц)

### Вариант B: Бесплатно на Render.com

1. https://render.com → New Web Service
2. Загрузи код, укажи `python bot.py` как команду запуска
3. Бесплатный тариф — работает без остановок

### Вариант C: На своём компьютере/сервере

```bash
pip install -r requirements.txt
python bot.py
```

---

## Для сотрудников

Каждый сотрудник должен:
1. Найти бота в Telegram по username
2. Нажать /start — один раз, чтобы бот запомнил их
3. Дальше бот сам напомнит в 21:10
